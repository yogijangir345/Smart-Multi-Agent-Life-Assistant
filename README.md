# 🤖 Smart Multi-Agent Life Assistant

A smart AI-powered personal assistant built using **Google Agent Development Kit (ADK)** and **Gemini 2.5 Flash**.
This project was created as my Capstone Project for the **5 Days AI Agents: Intensive Vibe Coding Course with Google**. The idea behind this project was to build a secure multi-agent assistant that can help users plan trips, manage budgets, remember preferences, and provide personalized assistance while keeping interactions safe.


# 📖 Project Overview

While learning about AI Agents during the course, I wanted to build something that could solve a real-world problem instead of creating a simple demo.
Travel planning is something many people struggle with. It usually involves searching multiple websites, calculating expenses, remembering previous choices, and organizing everything in one place.
To solve this, I built **Smart Multi-Agent Life Assistant**, where multiple AI agents work together to handle different responsibilities such as planning, budgeting, maintaining user context, and validating requests before generating responses.
